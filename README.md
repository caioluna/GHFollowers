# GitHub Followers

A project developed in Sean Allen's course using programmatic best practices that consists of an app to show GitHub's user data and followers.  


## Technology

**See below what we used in this project**:

- **Xcode:** Apple's official integrated development environment (IDE) for iOS app development.
- **Swift:** A powerful and intuitive programming language used to develop iOS apps.
- **UIKit:** Framework for building user interfaces.
- **View Code:** A programmatic approach to building user interfaces.


## Run the Project

**To run this project, you need:**

- To have macOS running on your machine.
- To have the Xcode IDE installed on your machine. You can download it from the App Store or from [Apple's developer site](https://apps.apple.com/us/app/xcode/id497799835?mt=12).
